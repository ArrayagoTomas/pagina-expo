Basurero Automatico

Paso a Paso

Planteamiento de la idea

El proyecto surge con el objetivo de automatizar la separación de residuos, facilitando el reciclaje mediante el uso de sensores y servomotores.
La idea principal fue diseñar un sistema con dos tachos:
Uno para residuos reciclables (bolsa verde).
Otro para residuos comunes (bolsa negra).
Cada tacho se abre automáticamente al detectar el color correspondiente de la bolsa.

Diseño y boceto

Antes de comenzar con la construcción, se realizaron bocetos en papel cuadriculado para definir:
La disposición de los tachos.
La ubicación de los sensores de color y de distancia.
El mecanismo de apertura de las tapas.
El espacio interno para la placa Arduino, los cables y la batería.
Este diseño permitió planificar la estructura en madera y prever los agujeros donde se colocarían los sensores.

Selección de componentes

Los principales componentes utilizados fueron:
Placa Arduino UNO.
Sensor TCS3200 (detección de color).
Sensor ultrasónico HC-SR04 (detección de distancia o movimiento).
Servomotores SG90 o MG90S (para apertura de las tapas).
Protoboard y cables Dupont.
Fuente de alimentación o batería de 9V.
Estructura de madera o plástico reciclado.

Montaje del circuito

Conectar el sensor de color al Arduino según sus pines (S0, S1, S2, S3, OUT, VCC, GND).
Conectar el sensor ultrasónico a los pines de disparo (Trig) y eco (Echo).
Conectar los servomotores a los pines PWM del Arduino (por ejemplo, 9 y 10).
Alimentar todo el circuito desde el Arduino o desde una fuente externa de 5V.
Probar las conexiones en una protoboard antes de soldar o montar definitivamente.

Construcción física

Cortar y armar la estructura de los tachos con madera o cartón duro.
Realizar los agujeros precisos para los sensores, cuidando la alineación con el área de detección.
Instalar los servomotores con soportes, asegurando que puedan mover las tapas sin trabarse.
Organizar los cables, evitando tensiones o cruces, y fijarlos para evitar desconexiones.

Programación del sistema

En el entorno del Arduino IDE, se cargó un programa que:
Lee la distancia con el sensor ultrasónico.
Cuando una bolsa se acerca (por ejemplo, a menos de 10 cm), el Arduino activa el sensor de color.
El sensor de color entrega valores RGB.
Según esos valores:
Si el color detectado se aproxima al verde, se activa el servo del tacho de reciclaje.
Si el color es negro o oscuro, se activa el servo del tacho de residuos comunes.
Después de unos segundos, las tapas se cierran automáticamente.

Calibración de sensores

Este paso fue crucial:
Se tomaron múltiples lecturas del sensor TCS3200 con diferentes colores y condiciones de luz.
Se registraron los valores RGB promedio para cada color de bolsa.
En base a esos valores, se definieron los rangos de detección en el código (por ejemplo, verde entre ciertos valores de G y B).
También se ajustó la distancia de activación del sensor ultrasónico para evitar que se active por error.

Pruebas y correcciones

Durante las pruebas, surgieron varios ajustes:
Problemas de calibración (colores muy similares).
Apertura inexacta del servo por la falta de bisagra.
Cables cortos o flojos, que interrumpían las señales.
Errores en librerías que impedían la lectura del color.
Todos estos inconvenientes fueron solucionados mediante ajustes mecánicos, eléctricos y de código.
